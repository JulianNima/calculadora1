# Calculadora en PSeInt

Proyecto final del curso de Programación I - Universidad Mariano Gálvez de Guatemala

## Descripcion

Aplicacion de calculadora desarrollada en pseudocodigo con PSeInt que ofrece 11 funciones distintas. Permite realizar operaciones aritmeticas basicas, manipular matrices, generar figuras geometricas y calcular estadisticas sobre listas de numeros.

## Que hace

- Operaciones aritmeticas: suma, resta, multiplicacion y division
- Operaciones con matrices: suma de elementos, resta de elementos, suma entre matrices, resta entre matrices
- Figuras geometricas: triangulo con numeros impares y rectangulo con asteriscos
- Analisis estadistico: suma, promedio, maximo, minimo, conteo de valores arriba/abajo del promedio
- Validaciones: control de division por cero, limite de 100 numeros, validacion de opciones
- Menu interactivo que vuelve al inicio despues de cada operacion

## Como ejecutar

1. Descargar e instalar PSeInt desde pseint.sourceforge.net
2. Abrir PSeInt
3. Archivo > Abrir y seleccionar ProyectoFinal.psc
4. Presionar F9 o clickear Ejecutar
5. Usar el menu ingresando el numero de la opcion deseada

## Requisitos

- PSeInt version 2024 o superior
- Windows, Linux o macOS
- 100 MB de espacio en disco

## Opciones del menu

1. Suma
2. Resta
3. Multiplicacion
4. Division
5. Sumar todos los valores de la matriz
6. Restar todos los valores de la matriz
7. Suma de matrices
8. Resta de matrices
9. Triangulo con numeros
10. Rectangulo con asteriscos
11. Estadisticas en una lista de numeros
12. Salir
